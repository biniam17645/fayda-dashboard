# Fayda ID Dashboard

A Telegram Mini App dashboard for the Fayda ID Converter Bot.

## Features

- 📊 Real-time ID statistics
- 💳 Buy credit packages (1, 3, 5, 10, 30 IDs)
- 📤 Upload ID images (Screenshot, Bulk, PDF)
- 📜 ID history and download
- 👥 Referral system
- 🎨 Template selection with previews
- 💰 Payment history
- 📱 Telebirr & CBE Birr payment integration

## Tech Stack

- HTML + CSS + JavaScript
- Telegram Web App SDK
- Cloudflare Pages (hosting)
- Turso Database (backend)

## Deployment

This is a static web app hosted on Cloudflare Pages. The backend API runs on a VPS.
